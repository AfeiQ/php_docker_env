# Usage

## Start 

Run follow command in this dir:

`./start.sh`

## Stop

Run follow command in this dir:

`./stop.sh`

## Restart

Run follow command in this dir:

`./restart.sh`


## Port info

`cat .env`

# Config

## crontab

Edit `php_cli_crontab/crontab`

## worker

Edit `php_cli_worker/worker.sh`
